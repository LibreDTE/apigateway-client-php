API Gateway es un proyecto que tiene por objetivo gestionar trámites del Servicio de Impuestos Internos (SII) de Chile.

Aquí podrás encontrar la biblioteca escrita en lenguaje PHP para la integración con el Servicio de Impuestos Internos (SII) asociada la creación de BHE, DTE, obtención de datos de contribuyente, etc.
